# 🏠 House Price Prediction using Machine Learnin

A Machine Learning web application that predicts house prices based on the area (in square meters).

---

## 📌 Project Overview

This project uses a Linear Regression model trained on historical house price data.

Users simply enter the house area, and the application predicts the estimated price instantly.

---

## 🚀 Features

- Predict house prices
- User-friendly Streamlit interface
- Input validation
- Error handling using try-except
- Saved ML model using Joblib
- Clean project structure

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Joblib

---

## 📂 Dataset

The dataset contains two columns:

| Area (sq meter) | Price |
|-----------------|-------|
| 80 | 4500000 |
| 95 | 5200000 |
| 110 | 6100000 |

---

## 📈 Model Used

Linear Regression

---

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/YourUsername/House-Price-Prediction.git
```

Move inside the folder

```bash
cd House-Price-Prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
